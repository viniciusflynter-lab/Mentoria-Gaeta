# Mentoria Gaeta - Landing Page

Esta é a landing page do projeto **Mentoria Gaeta - Maestria em Massoterapia**. O projeto consiste em uma página web (HTML, CSS via Tailwind CDN) totalmente otimizada e modular para alta conversão, mantendo uma aparência e design premium (*glassmorphism*, fontes serifadas e elementos dinâmicos).

## 🚀 Como subir e rodar
O projeto é puramente estático (Vanilla HTML + Tailwind CSS via CDN + Iconify).
Para rodá-lo localmente, basta abrir o arquivo `index.html` em qualquer navegador moderno. Não há necessidade de build, pois o Tailwind está importado como CDN para facilidade no desenvolvimento.

## 📁 Estrutura de Arquivos
- `/index.html`: O documento principal contendo todas as marcações estruturais, a Hero, e as dobras de conversão da Mentoria.
- `/assets/css/style.css`: Arquivo separado contendo as definições das fontes e estilos *custom* do *design system* da marca (efeitos *glass-panel*, sombras premium, background, etc.).

## 🌐 Deploy na Vercel
Este projeto já possui a estrutura otimizada e pronta para deploy na [Vercel](https://vercel.com).
Como é um projeto HTML estático de "pasta raiz" (`root directory`):
1. Suba todo este repositório para o seu GitHub.
2. Acesse a Vercel e clique em "Add New Project".
3. Importe o repositório do GitHub.
4. A Vercel detectará automaticamente que se trata de uma aplicação estática, e você só precisará clicar em "Deploy". Em segundos, o seu projeto estará no ar.
