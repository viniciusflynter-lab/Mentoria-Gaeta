O que você vai aprender na Trilha?

Você vai aprender a programar em Python do zero, entender os conceitos fundamentais da programação e aplicar tudo diretamente na criação de sites dinâmicos com Django.

Também vai dominar a estrutura de um projeto web, trabalhando com rotas, views, templates, banco de dados e versionamento com Git.

Faça deploy da sua aplicação na nuvem

Gerencie banco de dados e modelos

A trilha Python para Web é um programa de cursos que capacita você a construir aplicações web modernas com Python. Você aprende desde os fundamentos da programação até sistemas dinâmicos com Django e Flask, integração com bancos de dados, autenticação, deploy na nuvem e versionamento com Git — tudo para entregar sites e apps funcionais e prontos para o mercado.

Você passa a criar sites e sistemas dinâmicos que automatizam processos, oferecem interatividade e garantem boa experiência para os usuários. Python é uma das linguagens mais usadas no desenvolvimento web, e dominar Django e Git abre portas para carreiras promissoras e bem remuneradas.

Programação em Python do básico a aplicações web; criação de sites dinâmicos com Django; estruturação de rotas, views e templates; integração com bancos de dados como SQLite e PostgreSQL; versionamento e colaboração com Git e GitHub; e deploy de aplicações na nuvem com PythonAnywhere ou AWS.
